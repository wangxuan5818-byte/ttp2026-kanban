## 一、产品概述

TTP2026 战略看板是一套面向企业战略执行的**数字化指挥中心系统**，以"三体人突围战"为主题，帮助管理层实时掌握各部门战略任务进展、效益核算、积分排名和卡点预警。

### 核心价值

| 维度 | 描述 |
|------|------|
| **战略可视化** | 将年度战略目标分解到各部门任务，实时展示执行进度 |
| **效益量化** | 通过ROI核算模型，量化每项任务的AI赋能效益 |
| **卡点预警** | 自动识别超期、阻塞任务，推送钉钉通知 |
| **积分激励** | 部门积分排行榜，激励各部门主动更新任务进展 |
| **多端支持** | 网页端（浏览器访问）+ 桌面客户端（Electron） |

## 四、核心功能模块

### 4.1 战略总览（管理员视图）

- **年度战略目标**：展示战略目标、目标金额、截止日期、三条路径
- **任务统计卡片**：总任务数、进行中、有卡点、已结束
- **各部门任务进度**：每个部门的进度条、完成率、任务列表
- **战略路径里程碑**：三条路径的关键节点时间轴
- **全局效益汇总**：各部门效益数据汇总面板
- **卡点预警**：超期或被标记为卡点的任务列表

### 4.2 部门看板（部门成员视图）

- **部门信息头部**：部门名称、负责人、整体进度
- **任务卡片列表**：每个任务的状态、本周结果、进度
- **任务详情弹窗**：点击任务卡片展开详情，支持编辑
- **效益核算面板**：每个任务的ROI分析、场景对比、效益记录

### 4.3 效益核算（详细开发）

效益核算面板支持以下功能：

| 功能 | 说明 |
|------|------|
| **ROI仪表盘** | 年化总价值、平均ROI、效益等级（S/A/B/C/D） |
| **三类效益展示** | 时间节省、质量提升、成本降低的迷你图表 |
| **场景对比分析** | AI前后可视化对比，改善幅度百分比 |
| **效益记录卡片** | 每条记录的详细展示，支持编辑/删除 |
| **双视图切换** | 记录视图 + 分析视图 |
| **效益等级评估** | 根据ROI自动评定S/A/B/C/D等级 |

效益等级标准：

| 等级 | ROI范围 | 描述 |
|------|---------|------|
| S | ≥500% | 卓越效益，战略级价值 |
| A | 200-499% | 优秀效益，高价值任务 |
| B | 100-199% | 良好效益，正向回报 |
| C | 50-99% | 一般效益，需持续优化 |
| D | <50% | 待提升，需重新评估 |

## 七、数据模型

### 任务（Task）

```python
class Task(Base):
    id: str                    # 任务ID
    committee_id: str          # 所属部门ID
    name: str                  # 任务名称
    status: str                # 状态：进行中/已完成/暂缓/终止
    priority: str              # 优先级：high/medium/low
    owner: str                 # 负责人
    current_result: str        # 本周结果
    progress: int              # 进度（0-100）
    has_blocker: bool          # 是否有卡点
    blocker_description: str   # 卡点描述
    deadline: datetime         # 截止日期
    score: int                 # 积分
```

### 效益记录（Outcome）

```python
class Outcome(Base):
    id: int
    task_id: str               # 关联任务
    committee_id: str          # 所属部门
    scenario: str              # 应用场景
    time_saved_hours: float    # 节省时间（小时/月）
    quality_improvement: float # 质量提升（%）
    cost_reduction: float      # 成本降低（元/月）
    roi_percentage: float      # ROI（%）
    before_ai: str             # AI前描述
    after_ai: str              # AI后描述
    annual_value: float        # 年化价值（元）
```
